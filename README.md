# Client Calculation Server

Client-server calculator implementation using Java TCP, UDP, multithreading, and operation statistics.

## Features

* TCP-based arithmetic request handling
* UDP-based server discovery
* Concurrent client handling
* Periodic statistics reporting
* Support for ADD, SUB, MUL, and DIV operations
* Error handling for invalid requests

## Technologies

* Java
* TCP
* UDP
* Multithreading

## How to run

javac *.java

java CCS <port>
